# 簡易掲示板アプリ
	Pythonを使用し、Flaskで作った登録制の簡易掲示板です

# 主な機能
	-ユーザー名、パスワードを使用したユーザー登録(必須)
	-掲示板への投稿、およびコメント、いいね機能
	-投稿の編集、削除(自分の投稿のみ)
	-投稿の検索機能、および並び替え機能(新着順、投稿順、更新順、いいね数順)
	-ユーザー情報の編集(本人のみ)

# 使い方
	Flaskを導入したPython環境を利用して「app.py」を実行し、
	http://127.0.0.1:5000/ にアクセスしてください。

# 注意点
	本アプリは学習途上のものにより、セキリュティ対策が不十分です。
	現在はパスワードのハッシュ化のみが行われています。(順次改善予定)
	***また、本アプリのhtmlおよびCSS部分はChatGPTによるアシストを使用しました。***
